3. 세 실수 x, y, z가 1x+ 1y+ 1z= 12 과 2x=3y=5z를 만족시킬 때, 2x+3y+5z3의 값을 구하시오. [3.0점]
① 780 ② 810 ③ 840 ④ 870 ⑤ 900
2x = 3y = 5z = k 로 놓으면 k1/x=2,   k1/y=3,   k1/z=5 k1/x + 1/y + 1/z = 2×3×5 k1/2 = 30   →   k = 900 ∴ (2x+3y+5z)/3 = k = 900
4. 함수 y = 3x−a+b의 그래프가 다음 그림과 같을 때, a+b의 값은? (단, a, b는 상수이고 직선 y=−1은 점근선이다.) [3.0점]
① −6 ② −5 ③ −4 ④ −3 ⑤ −2
점근선이 y=−1 이므로   b = −1 →   y = 3x−a − 1 점 (−1, 0)을 지나므로:    0 = 3−1−a − 1   →   −1−a = 0   →   a = −1 ∴ a + b = −2
5. 2 ≤ n ≤ 10인 자연수 n에 대하여 n²+1의 n제곱근 중 실수인 것의 개수를 f(n), n²−8n+12의 n제곱근 중 실수인 것의 개수를 g(n)이라 하자. f(n) = 2g(n)을 만족시키는 모든 자연수 n의 값의 곱은? [5.0점]
① 6 ② 8 ③ 10 ④ 12 ⑤ 14
f(n) = { 1 (n이 홀수), 2 (n이 짝수) } f(n)=2g(n) 만족 → f(n)=2, g(n)=1 n이 짝수이므로 n²−8n+12 = 0 (n−2)(n−6) = 0   →   n = 2 또는 n = 6 ∴ 모든 자연수 n의 곱 = 2×6 = 12
