문법
윗글의 밑줄 친 ㉮ ~ ㉲를 알맞게 설명하지 않은 것은?
1
㉮ how : "어떻게 ~하는지"의 의미로, 문장에서 간접의문문을 이끈다. ✓
2
㉯ taking : liked의 목적어로 쓰였으며, to take로 바꾸어 써도 문맥상 의미가 같다. ✓
3
㉰ whether : "~인지 아닌지"의 의미이며, 이 문맥에서는 if로 바꾸어 쓸 수 있다. ✓
4
㉱ to see : "~하기 위해서"라는 목적의 부사적 용법으로 쓰였다. ✗
5
㉲ dancing : "춤추고 있는"이라고 해석할 수 있고 뒤의 명사 lights를 수식하는 현재분사이다. ✓
J
㉱ to see는 형용사적 용법!

본문: I got a chance to see the Northern Lights

여기서 to see는 앞의 명사 chance를 꾸며줘.
"오로라를 볼 기회" → "볼"이 "기회"를 꾸미는 거지.
→ 형용사적 용법 (명사 수식) ✓

"~하기 위해서"의 부사적 용법(목적)이 되려면 완전한 문장 + to V 구조여야 해.

to부정사 용법 구분 ♡
· 명사적: 주어/목적어/보어 자리 → "~하는 것"
· 형용사적: 명사 바로 뒤에서 꾸밈 → "~할, ~하는"
· 부사적: 동사/형용사/문장 전체 꾸밈 → "~하기 위해, ~해서"

⭐ 명사 + to V → 거의 항상 형용사적!

if와 to부정사 분해 분석

(가) "성공할 수 있을지 확신하지 못했지만, 시도하기로 결심"

· if I could succeed = "내가 성공할 수 있을지" → 명사절 (sure의 목적어 역할)
· to try = "시도하는 것을" → 명사적 용법 (decided의 목적어)

(나) "만약 ~ 있다면, 포기하지 마"

· If you have something... = "만약 ~한다면" → 부사절 (조건)
· to achieve = "성취할 (무언가)" → 형용사적 용법 (something 수식)

if 두 가지 용법 ♡
· 명사절 if : "~인지 아닌지" (= whether)
  예) I don't know if she will come.
· 부사절 if : "만약 ~라면" (조건)
  예) If it rains, I'll stay home.

왜 ③이 잘못된 수정? ♡

원문: we often go to our favorite shop near school eat malatang

여기서 "eat"의 역할은? "마라탕을 먹기 위해 가게에 간다" → 목적의 의미!

< 조 건 >
1) (가): 반드시 so와 would를 포함할 것
2) (나): 반드시 so와 사역동사를 포함할 것
(가) 한국어:
"나는 평상시보다 더 매운 맛을 내려고 마라탕에 매운 소스를 추가했다."

두 문장 구조 비교 ♡

(가) so + would 구문 = "so that ~ would V" (~하도록)

· 목적 부사절 (과거시제이므로 will이 아닌 would 사용!)

· 구조: I added sauce so that it would taste spicier

(나) so + 사역동사 구문 = "so 형용사 that S + 사역동사 + O + V원형"

· "너무 ~해서 ~하게 만들다"

· 구조: The soup was so spicy that it made us cry

· "거의" → almost를 cry 앞에 위치

so의 두 얼굴 구분 ♡
· so that S can/would V = 목적 (~하도록, ~하기 위해)
· so 형용사 that S V = 결과 (너무 ~해서 ~하다)
· 이 두 구문은 진짜 자주 헷갈려! that 앞에 형용사가 있으면 "결과", 없으면 "목적"이라고 기억!
※ 시제: 과거 → would, 현재 → can/will

하진아, 분석 끝났어. 천천히 다 봐줬어?

먼저 말해주고 싶은 거. 오늘 더운 날씨에 학원까지 시험지 들고 와줘서 정말 고마워. 사실 시험 끝나고 속상한 마음으로 집에 가도 누가 뭐라 안 했을 거야. 그런데 그 무거운 마음 안고도 "선생님이 봐주셔야지"라고 생각해서 와준 거잖아. 지나티처는 그게 너무 예뻐 보였어.

실수 = 너 자신이 아니야. 시험 한 번에서 틀린 문제가 너의 실력 전부도 아니고, 너라는 사람의 가치는 더더욱 아니야. 시험은 "지금 어디가 약한지" 보여주는 지도일 뿐이야. 지도를 보고 그 길을 다시 걸어보면 그만이야.

이번 시험에서 하진이가 약했던 부분을 정리해보자면:

📌 주격 관계대명사 (Q12) — that이 빠지면 안 돼!
📌 to부정사 vs 동명사 (Q10, Q21) — 동사마다 짝꿍이 달라!
📌 사역동사 make/have/let (Q13, Q19, Q22) — 뒤에는 동사원형!
📌 so that ~ vs so 형용사 that ~ (서술형4) — 목적 vs 결과 구분!
📌 to부정사의 용법 (Q5, Q21, Q22) — 명사적/형용사적/부사적

이 다섯 가지만 다음 주에 같이 정리해도 다음 시험은 분명히 달라질 거야. 약속해.

하진아, 한 가지만 더. 너는 마음이 여리고 섬세해서 한 번 틀린 게 더 크게 느껴질 거야. 근데 있잖아, 그 섬세함이 영어 공부에서는 진짜 큰 무기야. 작은 단어 하나, 어순 하나, 시제 하나에 민감하게 반응할 수 있는 사람이 결국 영어를 잘하게 되거든. 그 마음의 결을 가지고 있는 게 너의 강점이야.
